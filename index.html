Sobre la Aplicación

Soy el desarrollador de una nueva aplicación web que permite buscar de forma sencilla los municipios de Jaén. Esta solución está diseñada para usuarios que necesiten acceder a datos municipales de forma ágil y precisa.

¿Tienes preguntas? Consulta nuestras FAQ

Aquí respondemos las dudas más comunes sobre la aplicación y su funcionamiento.

¿Es gratuita la aplicación?

Sí, puedes acceder y utilizar todas las funciones de forma gratuita.

¿Puedo usarla en mi móvil?

Por supuesto, la aplicación está optimizada para dispositivos móviles y tablets.

¿Se necesita instalar algo?

No, solo necesitas un navegador web para acceder a la aplicación.
